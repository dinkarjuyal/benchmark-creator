# CDBench: Controlled Multi-Fault Debugging Benchmark

Code and data for the paper **"CDBench: Controlled Multi-Fault Debugging with Reinforcement Learning and Iterative Repair"**.

CDBench makes bug count $N$ a controllable experimental variable. We inject $N$ simultaneous corruptions into open-source library source code and measure per-group fix rate as $N$ scales from 1 to 10. Corruptions are generated by an adversarial three-player pipeline adapted from the SGS (Self-Guided Self-Play) framework.

## Repository Structure

```
paper/                        ← LaTeX source and figures
  main.tex
  references.bib
  figs/
    generate_figures.py       ← reproduces all paper figures
    degradation_curves.pdf
    iterative_repair.pdf
    repair_training.pdf
    pipeline.pdf

scripts/
  generate_sgs_vllm.py        ← SGS corruption generation (Proposer → Executor → Guide)
  functional_judge.py         ← evaluation judge (applies fixes, runs pytest)
  run_eval_local.py           ← baseline and RL model evaluation
  train_rl.py                 ← GRPO reinforcement learning training
  discover_tfail.py           ← discover T_fail sets for each corruption
  analyze_results.py          ← result analysis utilities
  generators/
    coding_diffusion.py       ← CorruptionSpec dataclass and generation utilities
```

## Requirements

```bash
pip install vllm torch transformers trl peft scikit-learn scipy pytest
```

GPU: 4× H100 (or equivalent) recommended for training. Evaluation can run on fewer GPUs.

## Generating Corruptions

```bash
python scripts/generate_sgs_vllm.py \
    --model Qwen/Qwen2.5-Coder-32B-Instruct \
    --library sklearn \
    --output results/sgs_corruptions_sklearn.json \
    --tp 4
```

The pipeline runs Proposer → Executor → Guide on each source file and outputs a JSON catalog of verified corruptions.

## Running Evaluations

```bash
python scripts/run_eval_local.py \
    --model Qwen/Qwen2.5-Coder-7B-Instruct \
    --corruption-file results/sgs_corruptions_sklearn.json \
    --n-values 1 3 5 \
    --output results/local_eval/7b_fs.json
```

## RL Training

```bash
python scripts/train_rl.py \
    --model Qwen/Qwen2.5-Coder-7B-Instruct \
    --corruption-file results/sgs_corruptions_sklearn.json \
    --output-dir checkpoints/rl_7b_fs \
    --steps 100 \
    --gpus 0,1,2,3
```
